GAS: Guidance on Application and Streaming（新高考录取及分流指导系统）

一：项目简介
我们的项目是一个针对高考志愿填报的推荐系统，主营方向为“招生信息+在读分享”一站式查询。
共两种查询方式：
（1）综合查询
通过输入想要查询的内容（学校名、专业、分数、城市、宿舍、伙食、保研等），对于招生计划、录取分数线、在读体验以及志愿搭配进行查询（根据当年官方招生简章）
（2）选科查询
通过填入对应省份（可选）、选科组合、以及想要查询的内容，对于包括专业限报、科目要求、选课组合与能报的专业，进行查询（同样根据当年的官方招生简章）
同时，我们也提供了根据输入自己的信息（包括但不限于生源地、高考成绩、位次、科类等）来深化查询强度的服务，用户可以在‘我的信息’栏目输入自己相关信息实现查询目的。
同样地，已入学的同学同样可以通过这个平台为学弟学妹们提供宝贵的参考意见。同学们可以通过匿名分享就读的学校、城市、专业等信息，分享自己对于院校的各方面感受。同时为了确保信息的真实性与准确性，在分享时同学们需要上传自己的学信网学籍截图（必选项，但不会公开以保障隐私）。而后同学们可以通过‘查看分享’栏目查询学长学姐们提供的信息。
二：业务价值
我们的项目聚焦于当下学生们最关心的事件之一--高考志愿填报。
众所周知，志愿填报的选择明智与否同样是同学们所最需要关心的问题之一，而对于这个问题，虽然发达的网络早已给出解决方案，但各种信息实在太过繁杂，褒贬不一的评价更是让同学们在选择时感到茫然而无所适从。
为了对这个问题提供一些全新的解决思路，我们将目光投向AI，希图搭建一个一站式的咨询平台，为同学们提供一个更加便捷且具有相当的可信度的工具，辅助同学们更明智地填报志愿，助力同学们能够真正选到自己心仪的学校、心仪的专业。
三：AI创新性
1．个性化推荐引擎：并非简单的分数线匹配，而是从多个维度动态分配，基于同学们的相关需求对各类信息加权进行匹配。
2．动态预测与风险评估：利用机器学习等技术评估当年分数线波动、热点变化，以防止考生填报“扎堆”现象的出现导致同学们错过心仪的大学甚至于“滑档”
3．数据整合能力：将分散的数据集中统一，更加便于查询；同时根据AI预测用户偏好提供更加精确的、顺应同学需求的推荐算法。
四：技术实现说明
整个项目的实现分为两部分：AI查询和在读生信息收集。
AI查询部分，我们介入的是开源的Lama3模型，通过anythingLLM进行模型的训练。在目前阶段，由于准备的时间有限，我们通过自己编写的网络爬虫爬取了宁夏回族自治区2025年高考相关通告和高校招生信息，并且将信息用于AI的训练。未来我们会扩展我们的数据库，直至覆盖各个省份。
在读生信息收集部分，我们收集用户输入的信息，并且分类存储于mysql数据库中，调用时则从mysql数据库中直接调用。同时，数据库的应用也使得上传信息更易于管理。但是由于时间有限，此最初版本还未涵盖信息审核等重要部分，依然有待完善。
